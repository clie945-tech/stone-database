編號 控制項目 規範要求 實作現況 評估結果

A-5 多因子驗證 (MFA) 具特權帳號宜啟用 MFA 未實作，僅單因子（密碼） 尚未實施
A-6 權限分層 應依職務區分權限 管理員 / 會員 / 訪客三層完全分離，Session 隔離 已符合
A-7 Session 管理 Session Cookie 應設 Secure、HttpOnly、SameSite 未明確設定（依 Flask 預設） 部分符合
