Feature 01 · 核心

材質模擬 Material Simulation

上傳設計圖或空間照片，圈選要替換的區域（牆面、桌面、家具表面），AI 把所選循環石材的真實材質貼上去，即時預覽。

功能
  • 筆刷塗選 + 拖放上傳（支援 HEIC）
  • 主力引擎：Gemini 參考圖編輯，直接以實際石材照片為依據
  • 後備引擎：Flux Fill Pro 文字 inpainting
  • 結果自動存入「我的設計」、可產生提案 PDF
規格
  • 模型：gemini-2.5-flash-image
  • 後備：flux-fill-pro
  • 成本：約 NT$1.2 / 張（Gemini）
  • 權限：僅限登入會員（保護運算資源）
Feature 02 · 核心

牆面模擬 Wall Simulator

點選牆面四個角、輸入牆面實際尺寸，系統以幾何運算把石材精準貼合——透視與比例皆為數學精準，並同步顯示用量、庫存與減碳。

功能
  • 四角點選（支援觸控、可拖曳微調）
  • 透視投影貼合 + 依實際尺寸計算拼貼片數
  • 棋盤鏡像對花、可選拼縫線、原牆光影轉移
  • 同步顯示：拼貼計畫 / 庫存檢查 / 減碳效益
規格
  • 引擎：本地幾何運算（Pillow + NumPy）
  • 成本：免費・即時出圖（不呼叫外部 AI）
  • 選配：Gemini AI 擬真強化（光影／遮擋還原）
  • 適用：平整牆面、桌面、地板
Feature 03 · 核心

草圖渲染 Sketch Render

上傳手繪草圖或概念線稿，選一塊循環石材，補一句產品描述，AI 把草圖造型 + 真實石材材質生成寫實產品渲染圖——概念秒變提案。

功能
  • 草圖（手繪/線稿）+ 石材 + 文字描述
  • 免遮罩：整張草圖即產品造型
  • 輸出工作室等級產品渲染（可當型錄/提案主圖）
  • 自動存入「我的設計」、可出提案 PDF
規格
  • 模型：gemini-2.5-flash-image
  • 成本：約 NT$1.2 / 張
  • 輸入：JPG / PNG / HEIC / WebP
  • 權限：僅限登入會員
Supporting · 輔助引擎

讓設計更聰明的底層 AI

Vision

視覺辨識

AI「看」石材照片，自動產生精準材質描述（底色、紋路、表面處理），並快取重用。

  • gemini-2.5-flash
  • 後備：LLaVA v1.6
Search

以圖搜圖

上傳參考圖，用感知雜湊（pHash）+ HSV 色彩比對，找出資料庫中最相似的循環石材。

  • 本地運算・免費
  • ImageHash + 色彩分析
Prompt

三層 Prompt 工程

管理員自訂 ＞ 視覺 AI 自動描述 ＞ 模板後備，確保每塊石材都有可用的高品質生成指令。

  • 自動快取至資料庫
  • 設計師可即時微調
Value-add · 加值串接

從「看效果」到「可成案」

減碳效益

依體積 × ICE Database v3.0 碳係數，自動計算每案減碳量與等效（樹／車）。

提案 PDF

一鍵產出含效果圖、規格、減碳、供應商的 A4 提案，內嵌中文字型。

我的設計

每位會員自動保留最近 30 張合成作品，回顧設計發展。

展覽圖庫

後台用 Gemini 批次生成「石材 × 情境」展示圖，供展覽使用。

Strategy · 成本與合規

務實的資源控管

4 / 7

逾半數引擎為本地或免費運算（牆面模擬、以圖搜圖、色彩偵測等）。

會員制

AI 工具僅限登入會員，保護 API 運算資源、避免成本暴衝。

無中資

技術堆疊皆為非中資來源（Google / BFL / 開源），符合資安規範。

主力以 Gemini，失敗自動退回 Flux；牆面模擬走本地運算。穩定、低成本、可隨 AI 技術演進同步升級。
