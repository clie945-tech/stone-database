## 沒放截圖會怎樣？

每個 `<img>` 有對應的 CSS 占位框，看起來像「待截圖」的設計稿，仍可正常播放影片。

## 截圖技巧

**Mac**：
- ⌘ + Shift + 4 → 框選區域截圖
- 建議在 Chrome / Safari 全螢幕下截圖
- 使用 1.5x 或 2x 解析度（Retina）效果最佳

**進階**：使用 Chrome DevTools 的 Device Toolbar 設定固定 1600×900 視窗截圖
