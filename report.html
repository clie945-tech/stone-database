01執行摘要

本平台不是「掛一個 AI 按鈕」的資料庫，而是讓 AI 深度滲透每個工作流關鍵節點。

04 · AI 視覺分析

04AI #2 — 視覺分析（讓系統「看懂」石材）

用 LLaVA v1.6 視覺語言模型，自動為每張石材照片產生精準的英文描述 prompt，作為設計合成的「燃料」。

運作原理

# 上傳一張蛇紋石照片 input: "gray_serpentine_004.jpg" ↓ LLaVA v1.6 Mistral 7B（視覺語言模型） output: "gray serpentine with darker veining, polished glossy surface" # 自動存入資料庫的 ai_prompt 欄位

這段描述自動存入資料庫的 ai_prompt 欄位，成為設計合成工具的精準引導。比起人工撰寫，AI 描述更貼近實際照片內容；比起單純的模板套用，AI 能感知細節（紋路走向、光澤度、色澤分布）。

三種觸發機制

① 自動執行
新增石材時自動觸發；主照片變更時重新分析。
② 手動觸發
編輯頁的「✨ AI 重新分析」按鈕，可即時更新單筆。
③ 批次升級
後台首頁的「✨ 重新分析 Prompt」可一鍵升級所有舊資料。

為何選 LLaVA

模型優點缺點
LLaVA v1.6 (本平台選用)跑在 Replicate 上、開源、便宜（NT$0.5/張）、足夠應付描述任務中文較弱（但我們用英文 prompt）
GPT-4 Vision中英都強、推理能力強需另開 OpenAI 帳號、貴 4–10 倍
Claude Sonnet Vision質感最好、推理最強需另開 Anthropic 帳號、貴 5 倍
設計決策
對於「看石材照片寫關鍵字」這種定義明確的任務，LLaVA 性價比最高。若未來要做「看設計圖自動建議搭配的石材」這類需要推理的功能，再升級到 Claude / GPT-4V。

07 · Prompt 工程

07AI #5 — 三層 Prompt 工程

每塊石材的 AI Prompt 不是黑盒子。管理員能看、能改、能還原；設計師甚至能即時微調當次合成。

套用優先順序

1
管理員手動編輯
最高優先級。對重要石材人工撰寫精準 prompt，確保關鍵商品的合成品質。
2
LLaVA 視覺 AI 自動生成
未手動編輯時，AI 看照片寫描述。大幅減輕管理員負擔，新增即用。
3
模板規則生成
AI 服務不可用時的安全網。依「種類 + 顏色 + 拋光」組合最簡關鍵字。
第四層：設計師端即時微調
設計師在設計合成工具中還能即時編輯當次的 prompt（不影響資料庫設定），達到 4 層彈性。「可控、可改、可審計」是此架構的核心精神。

範例：同一塊石材的三層輸出

層級輸出
① 人工honed Italian carrara marble, fine gray veining on warm white base, matte finish
② AIwhite marble with subtle gray veining, polished glossy surface
③ 模板white marble, polished surface

08 · 語義搜尋

08AI #6 — 智能語義搜尋

設計師輸入任一語言的查詢詞，系統自動展開同義詞庫，跨中英文匹配。

中英文同義詞自動展開

查詢詞自動展開查詢
大理石marble、雲石、大理岩、marmo
花崗岩granite、花崗石、麻石
蛇紋石serpentine、serpentinite
化石fossil、fossiliferous
marble大理石、雲石、大理岩
granite花崗岩、花崗石、麻石

為何不用 Embedding 向量搜尋？

設計決策
石材種類有限（4 大類）、查詢模式可預期、追求極致回應速度。同義詞字典在這個 scope 反而比向量搜尋更精準、更快、更易維護。

若未來品項擴展到上千種跨材質（木材、皮革、磁磚...），再升級到 Embedding-based semantic search 也不遲。

技術實作

  • 純 Python 實作，無需任何 ML 模型
  • 零延遲（記憶體字典 O(1) 查找）
  • 零隨用成本
  • 易維護：新增同義詞只需編輯 SYNONYMS 字典
  • 搜尋邏輯：原查詢詞 + 所有展開詞，用 OR 條件 SQL LIKE 比對

為何鎖住 AI 設計合成？

AI 設計合成每次成本 NT$1.5，若不限制權限，匿名訪客可能濫用 API 造成費用失控。透過會員制：

  • 使用者需先註冊（留下 Email、姓名）— 提高濫用成本
  • 可追蹤每位會員使用量 — 異常時可介入
  • 未來可加上「每會員每日上限」進一步保護
  • 後端 API 雙重防護：即使前端按鈕被 bypass，後端也擋下未登入請求

11傳統網站 vs. ReStone

同樣的石材，不同的呈現方式，產生截然不同的商業價值與使用體驗。

❌ 傳統石材網站

  • 商品圖 + 規格表 + Excel
  • 找料 → 拍照 → PS 半小時 → 寄圖給客戶
  • 業主、設計師、客戶各自想像
  • 純文字搜尋，搜「marble」找不到
  • 客戶看不到實際應用，難下決定
  • 剩料、循環石材難曝光
  • 無法跨域應用（家具、產品設計師難用）
  • 溝通靠 LINE 截圖 + 描述

✅ ReStone 平台

  • 商品圖 + AI 描述 + AI 設計合成 + 多軸搜尋
  • 找料 → 圈選 → 30 秒 AI 合成 → 即時給客戶
  • 同一張視覺化圖三方溝通
  • 中英文同義詞 + 以圖搜圖，搜「marble」也行
  • 客戶能看到家裡牆面換上後的樣子
  • 循環剩料能精準曝光在對的設計案中
  • 跨域應用：室內、家具、產品設計師都能用
  • 視覺化提案直接做進報告

15 · 路線圖

15未來 AI 升級方向

基礎已建好，接下來可漸進式加上更深的 AI 能力。

短期0–3 個月

  • SAM 自動辨識：設計圖上傳後，AI 自動框出「這是牆面、那是桌面」，使用者直接點選，不用畫筆刷
  • 合成歷史紀錄：會員專屬「我的合成」頁，可重複查看、下載、分享給客戶
  • 並排比對模式：同一張圖同時套 3 塊不同石材並排顯示，客戶直接選最喜歡的
  • 合成 PDF 報告：把「原圖 → 圈選 → 合成」做成一張 PDF 設計提案

中期3–9 個月

  • Claude / GPT 風格推薦：客戶上傳「想要的風格圖」→ LLM 推薦 5 塊最適合的石材並解釋為什麼
  • 多角度合成：客戶給多張照片（正面、側面、細節），AI 確保同一塊石材跨視角一致
  • 業主分析儀表板：哪塊石材被合成最多次、哪個顏色最熱、轉換率分析
  • 會員配額系統：每月免費 N 次合成，超過後需訂閱

長期9 個月以上

  • 跨材質 SaaS：同樣模板複製到木材、皮革、磁磚、布料，建立循環建材生態系
  • 3D 視角合成：不只 2D 照片，3D 模型也能套材質
  • AR 即時預覽：手機相機對著牆面 → 即時看到不同石材的效果
  • 專業版訂閱：為設計事務所提供團隊協作、品牌客製化、API 接入
當前狀態
所有基礎功能已上線並運作中。下一步重點：累積真實設計師使用數據，根據實際工作流痛點選擇優先項目。
